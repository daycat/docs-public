# 在VPS上安装Cloudflare Warp

Cloudflare Warp 为非双堆栈 VPS 提供了一种方便的方式来访问否则他们无法访问的内容。

# FsCarmen 脚本

wget -N https://cdn.jsdelivr.net/gh/fscarmen/warp/menu.sh && bash menu.sh

这将给您一个易于使用的界面，用于设置您的Warp连接以及其他一些有用的功能。

您可以在GitHub (opens new window)上查看Fscarmen的项目以及进一步的说明。

# P3terX的脚本

bash <(curl -fsSL git.io/warp.sh) [子命令]
子命令 用法
install 安装Cloudflare WARP官方Linux客户端
uninstall 卸载 Cloudflare WARP Official Linux Client
restart 重新启动Cloudflare WARP Official Linux Client
proxy 启用WARP客户端代理模式（默认SOCKS5端口：40000）
unproxy 禁用WARP客户端代理模式
wg 安装WireGuard和相关组件
wg4 配置WARP IPv4全球网络（含WireGuard），所有IPv4出站数据都通过WARP网络进行
wg6 配置WARP IPv6全球网络（含WireGuard），通过WARP网络的所有IPv6出站数据。
wgd 配置WARP双栈全球网络（含WireGuard），所有通过WARP网络的出站数据。
wgx 配置WARP非全球网络（含WireGuard），设置fwmark或接口IP地址以使用WARP网络。
rwg 重新启动WARP WireGuard服务
dwg 禁用WARP WireGuard服务
status 打印状态信息
version 打印版本信息
help 打印此信息或给定子命令的帮助
menu 中文特殊功能菜单

这是一个具有基本功能的脚本，因此也使其容易集成到脚本和工作流程中。

您可以在GitHub (opens new window)上查看P3terX的项目以及进一步的说明。
